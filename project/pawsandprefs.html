Paws & Preferences:
Find Your Favourite Kitty

Project Overview

Paws & Preferences is a playful Tinder-inspired web application that allows users to discover their feline preferences by swiping through an endless feed of random cats. Built with vanilla JavaScript, it replicates the core mechanics of modern dating apps including swipe gestures and instant feedback, to create an engaging and humorous experience. The web app connects to the CATAAS API to source its images and provides users with lighthearted personality insights based on their swiping patterns, and then showcasing clean front-end development and API integration.

Key Features

Intuitive Swipe Mechanics

Implemented a familiar swipe-to-like (right) and swipe-to-dislike (left) interface that works seamlessly on both touch and desktop devices.

Dynamic Content & Feedback

Fetches random cat images on-demand from an external API and generates humorous, which personalized results based on user choices.

Fully Responsive Design

Ensured a smooth and consistent user experience across mobile and desktop browsers through responsive CSS and cross-platform gesture handling.

Challenges & Solutions

Identified Challenges

Implementing reliable cross-platform swipe detection for both touchscreen and mouse input.

Creating a smooth and realistic card swipe animation that mimics native app feeling.

Managing application state and UI updates efficiently with vanilla JavaScript without a framework.

Solutions & Action Plan

Integrated the Hammer.js library to abstract complex gesture logic, which providing a unified API to handle swipe events consistently across all devices.

Implemented CSS transitions and transforms manipulated by JavaScript to animate the card's position and rotation in real-time based on drag velocity and direction.

Structured the code into modular functions to handle distinct tasks including API calls, DOM updates, game logic, which ensuring maintainability and preventing spaghetti code.
