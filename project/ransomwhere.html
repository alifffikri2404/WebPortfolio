RansomWhere?

Project Overview

RansomWhere? is a mobile party game designed to overcome the rising global threat of ransomware by transforming cybersecurity education into an engaging social experience. This game was developed for Hilti employees to raise awareness about ransomware attacks and also implemented prevention strategies through interactive gameplay, without requiring any prior IT knowledge. It addresses the critical need for proactive defense by leveraging the principle that people are 68% more productive when having fun according to the study, which making complex security concepts more accessible and memorable.

The game also immerses players in a world where they must complete 3 missions, find and interact with objects while solve object-specific trivia to learn about the fundamentals and consequences of ransomware. This game provided a flexible and scalable solution for organizations since it came as single-player and offline mobile application, to educate their entire workforce and turned mandatory training into an enjoyable activity which can reduces human risk factors.

Key Features

Educational Party Gameplay

Missions and trivia seamlessly integrate ransomware awareness into a fun and social party game format that suitable for all skill levels.

Offline Accessibility

Fully functional without an internet connection which allows employees to learn at anytime, anywhere, without security concerns.

Customizable Avatar System

Player choice of male or female avatars increases relatability and engagement with the game's educational content.

Challenges & Solutions

Identified Challenges

Making complex ransomware concepts understandable and engaging for non-technical employees.

Designing for a wide range of players with varying tech proficiency within a single organization.

Managing game data and player progress locally on a mobile device without online sync.

Solutions & Action Plan

Implemented relatable scenarios, interactive object discovery and trivia pop-ups to break down information into digestible, fun gameplay chunks.

Adopted a simple party game format with intuitive mechanics such as tap-to-interact and set clear missions, which eliminates complex controls and player elimination.

Used a lightweight SQLite database to efficiently store player preferences, progress and game state reliably on the device.
